# Analiza danych scoutingowych

Folder zawiera analizy wspierające proces scoutingu zawodników.

Celem analiz nie jest zastąpienie obserwacji live, lecz wsparcie procesu decyzyjnego poprzez dane.

## Plan rozwoju

### Etap 1

* profile zawodników,
* porównania pozycyjne,
* podstawowe statystyki.

### Etap 2

* percentyle pozycyjne,
* radar zawodnika,
* porównania do ligi.

### Etap 3

* scoring zawodnika,
* modele potencjału,
* wspieranie rekomendacji scoutingowej.

## Główna zasada

Dane wspierają decyzję scouta — nie zastępują obserwacji meczu.
