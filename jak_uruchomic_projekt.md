# Jak uruchomić projekt

## 1. Sklonuj repozytorium

```bash
git clone LINK_DO_REPO
```

## 2. Przejdź do folderu projektu

```bash
cd football-youth-scouting-framework
```

## 3. Zainstaluj biblioteki

```bash
pip install pandas matplotlib
```

## 4. Uruchom pierwszy ranking zawodników

```bash
python skrypty_python/01_ranking_zawodnikow.py
```

## 5. Uruchom wykres ocen zawodników

```bash
python skrypty_python/02_wykres_ocen.py
```

## Co dalej?

Projekt będzie rozwijany o:

* porównania pozycyjne,
* modele potencjału,
* radar zawodnika,
* dashboard scoutingowy,
* wspieranie decyzji scoutingowych przez dane.
